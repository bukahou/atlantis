# CSS 知識体系

## 核心概念

CSS はウェブページのスタイルを定義する言語で、モダン CSS は強力なレイアウトとアニメーション機能を提供します。

### モダン CSS 機能

```
レイアウトシステムの進化
┌────────────────────────────────────────┐
│  Float   →  Flexbox  →  Grid          │
│  従来レイアウト  1次元レイアウト  2次元レイアウト │
└────────────────────────────────────────┘

Flexbox vs Grid
┌─────────────────┐  ┌─────────────────┐
│    Flexbox      │  │      Grid       │
│  ┌───┬───┬───┐  │  │  ┌───┬───┬───┐  │
│  │ 1 │ 2 │ 3 │  │  │  │ 1 │ 2 │ 3 │  │
│  └───┴───┴───┘  │  │  ├───┼───┼───┤  │
│   → 1次元配列    │  │  │ 4 │ 5 │ 6 │  │
│   → 柔軟な分配   │  │  └───┴───┴───┘  │
└─────────────────┘  │   → 2次元グリッド │
                     │   → 精密な制御    │
                     └─────────────────┘
```

## 学習パス

### 1. レイアウト基礎

- **Flexbox** - フレキシブルボックスレイアウト
- **Grid** - グリッドレイアウト
- **ポジショニング** - relative, absolute, fixed, sticky

### 2. モダン機能

- **CSS 変数** - カスタムプロパティ
- **メディアクエリ** - レスポンシブデザイン
- **コンテナクエリ** - コンポーネントレベルレスポンシブ

### 3. ツールとフレームワーク

- **Tailwind CSS** - アトミック CSS
- **CSS Modules** - モジュール化スタイル
- **プリプロセッサ** - Sass, Less

## ベストプラクティス

1. **モバイルファースト** - 小さい画面から設計開始
2. **セマンティック命名** - BEM またはアトミック
3. **変数使用** - デザイントークンを統一
4. **パフォーマンス** - 複雑なセレクタを避ける
