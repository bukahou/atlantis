# Kubernetes 知識体系

## 核心概念

Kubernetes (K8s) はコンテナオーケストレーションのデファクトスタンダードで、コンテナ化アプリケーションの自動デプロイ、スケーリング、管理機能を提供します。

## 学習パス

### 1. 核心リソース

- **Pod** - 最小スケジュール単位
- **Deployment** - ステートレスアプリケーション管理
- **Service** - サービスディスカバリとロードバランシング
- **ConfigMap/Secret** - 設定管理

### 2. 高度なオブジェクト

- **StatefulSet** - ステートフルアプリケーション
- **DaemonSet** - ノードレベルデーモン
- **Job/CronJob** - バッチ処理タスク
- **Ingress** - HTTP ルーティング

### 3. 運用管理

- **Helm** - パッケージマネージャー
- **Kustomize** - 設定管理
- **Prometheus** - 監視アラート

## ベストプラクティス

1. **リソース制限** - requests と limits の設定
2. **ヘルスチェック** - liveness と readiness の設定
3. **ローリングアップデート** - ゼロダウンタイムデプロイ戦略
4. **Namespace 分離** - マルチ環境リソース分離
