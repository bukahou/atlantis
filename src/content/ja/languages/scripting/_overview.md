# スクリプト言語知識体系

## 核心概念

スクリプト言語はその柔軟性と高速開発能力で知られ、自動化、データ処理、Web 開発に広く使用されています。

### スクリプト言語の特徴

```
特性の優位性
┌─────────────────────────────────────┐
│  高速開発  │  コード簡潔、迅速なイテレーション │
├────────────┼────────────────────────┤
│  動的型付け │  高い柔軟性、コンパイル不要   │
├────────────┼────────────────────────┤
│  豊富なエコ │  ライブラリとフレームワークが豊富 │
├────────────┼────────────────────────┤
│  クロスプラット│  一度書けばどこでも動く     │
└────────────┴────────────────────────┘
```

## 学習パス

### Python

- **基本構文** - データ型、制御フロー、関数
- **オブジェクト指向** - クラス、継承、マジックメソッド
- **高度な機能** - デコレータ、ジェネレータ、コンテキストマネージャ
- **エコ応用** - Web (Django/Flask)、データ (Pandas/NumPy)

### Shell/Bash

- **基本コマンド** - ファイル操作、テキスト処理
- **スクリプト** - 変数、条件、ループ
- **高度なテクニック** - パイプ、リダイレクト、プロセス制御
- **実践応用** - 自動化スクリプト、運用ツール

## ベストプラクティス

1. **Python** - PEP8 遵守、仮想環境使用
2. **Shell** - set -e 使用、エラーハンドリング追加
3. **共通点** - 明確なコメント、モジュール化設計
