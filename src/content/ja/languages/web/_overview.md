# Web 言語知識体系

## 核心概念

JavaScript は Web 開発の核心言語で、TypeScript は型システムを追加することで大規模プロジェクトの保守性を向上させます。

### 言語の進化

```
JavaScript 進化の歴史
┌─────────────────────────────────────────────┐
│ ES5 (2009)    │ 厳格モード、配列メソッド      │
├───────────────┼─────────────────────────────┤
│ ES6 (2015)    │ クラス、モジュール、アロー関数、Promise │
├───────────────┼─────────────────────────────┤
│ ES7-ES12      │ async/await、スプレッド演算子 │
├───────────────┼─────────────────────────────┤
│ TypeScript    │ 型システム、インターフェース、ジェネリクス │
└───────────────┴─────────────────────────────┘

TypeScript の価値
┌──────────────────────────────────────────────┐
│  型安全性    →  コンパイル時にエラー発見      │
│  IDE サポート →  より良い自動補完             │
│  リファクタ力 →  安全なコードリファクタリング  │
│  ドキュメント →  型はドキュメント             │
└──────────────────────────────────────────────┘
```

## 学習パス

### JavaScript

- **基本構文** - データ型、関数、スコープ
- **ES6+ 機能** - 分割代入、モジュール、クラス
- **非同期** - Promise、async/await
- **DOM/BOM** - ブラウザ API

### TypeScript

- **型基礎** - 基本型、ユニオン型
- **高度な型** - ジェネリクス、マップ型
- **型ユーティリティ** - Partial、Pick、Omit
- **エンジニアリング設定** - tsconfig、モジュール解決

## ベストプラクティス

1. **TypeScript** - strictモード、any を避ける
2. **非同期処理** - async/await 優先
3. **モジュール化** - ES Modules 規範
4. **コード品質** - ESLint + Prettier
