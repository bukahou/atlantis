# Kubernetes 知识体系

## 核心概念

Kubernetes (K8s) 是容器编排的事实标准，提供自动化部署、扩展和管理容器化应用的能力。

## 学习路径

### 1. 核心资源

- **Pod** - 最小调度单元
- **Deployment** - 无状态应用管理
- **Service** - 服务发现与负载均衡
- **ConfigMap/Secret** - 配置管理

### 2. 高级对象

- **StatefulSet** - 有状态应用
- **DaemonSet** - 节点级守护进程
- **Job/CronJob** - 批处理任务
- **Ingress** - HTTP 路由

### 3. 运维管理

- **Helm** - 包管理器
- **Kustomize** - 配置管理
- **Prometheus** - 监控告警

## 关键技能

| 技能领域 | 核心内容 | 应用场景 |
|---------|---------|---------|
| 资源管理 | YAML, kubectl | 日常运维 |
| 调试排错 | describe, logs, exec | 问题定位 |
| 扩缩容 | HPA, VPA | 弹性伸缩 |
| 安全 | RBAC, NetworkPolicy | 权限控制 |

## 最佳实践

1. **资源限制** - 设置 requests 和 limits
2. **健康检查** - 配置 liveness 和 readiness
3. **滚动更新** - 零停机发布策略
4. **命名空间隔离** - 多环境资源隔离
