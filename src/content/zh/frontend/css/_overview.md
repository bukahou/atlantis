# CSS 知识体系

## 核心概念

CSS 是定义网页样式的语言，现代 CSS 提供了强大的布局和动画能力。

### 现代 CSS 特性

```
布局系统演进
┌────────────────────────────────────────┐
│  Float   →  Flexbox  →  Grid          │
│  传统布局    一维布局     二维布局      │
└────────────────────────────────────────┘

Flexbox vs Grid
┌─────────────────┐  ┌─────────────────┐
│    Flexbox      │  │      Grid       │
│  ┌───┬───┬───┐  │  │  ┌───┬───┬───┐  │
│  │ 1 │ 2 │ 3 │  │  │  │ 1 │ 2 │ 3 │  │
│  └───┴───┴───┘  │  │  ├───┼───┼───┤  │
│   → 一维排列     │  │  │ 4 │ 5 │ 6 │  │
│   → 弹性分配     │  │  └───┴───┴───┘  │
└─────────────────┘  │   → 二维网格     │
                     │   → 精确控制     │
                     └─────────────────┘
```

## 学习路径

### 1. 布局基础

- **Flexbox** - 弹性盒布局
- **Grid** - 网格布局
- **定位** - relative, absolute, fixed, sticky

### 2. 现代特性

- **CSS 变量** - 自定义属性
- **媒体查询** - 响应式设计
- **容器查询** - 组件级响应

### 3. 工具与框架

- **Tailwind CSS** - 原子化 CSS
- **CSS Modules** - 模块化样式
- **预处理器** - Sass, Less

## 关键技能

| 技能领域 | 核心内容 | 应用场景 |
|---------|---------|---------|
| 布局 | Flex, Grid | 页面结构 |
| 响应式 | 媒体查询, clamp() | 多端适配 |
| 动画 | transition, animation | 交互体验 |
| 主题 | CSS 变量, dark mode | 个性化 |

## 最佳实践

1. **移动优先** - 从小屏幕开始设计
2. **语义化命名** - BEM 或原子化
3. **变量使用** - 统一设计 token
4. **性能优化** - 避免复杂选择器
